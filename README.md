# ✨ Crie seu Pequeno NPC — Sign Up UI

Este projeto faz parte do desafio **Daily UI**, onde o objetivo é criar interfaces diariamente para evoluir no design e desenvolvimento.

---

## Desafio
**Prompt:** Sign Up  
Criar uma interface de cadastro de forma criativa e funcional.

---

## Conceito
Transformei um formulário de cadastro em uma experiência interativa inspirada na criação de personagens (NPCs).

A proposta foi sair do padrão comum e criar algo mais envolvente, onde o usuário sente que está entrando em um universo mágico.

---

## Destaques do projeto
- Interface fofa e minimalista  
- Estética clean com elementos mágicos  
- Sistema de escolha de “essência” interativa  
- Mudança dinâmica de cores e barras (mana e energia)  
- Animações suaves no background  
- Feedback visual ao enviar o formulário  

---

## Tecnologias utilizadas
- HTML5  
- CSS3  
- Bootstrap 5  
- Bootstrap Icons  
- JavaScript  

---

## Funcionalidades
- Formulário de cadastro estilizado  
- Seleção de essência com interação visual  
- Barras que mudam conforme a escolha  
- Mudança de cores dinâmica  
- Popup de confirmação ao enviar  

---

## Objetivo
Praticar UI Design, melhorar a consistência visual e transformar interfaces simples em experiências mais criativas.

---

💖 Projeto desenvolvido como parte de um desafio pessoal de evolução em design e front-end.
